# Personal Habit Tracking – Data Science Project

## Goal
To analyse relationships between sleep, exercise, mood, and focus using statistical and machine learning techniques.

## Objectives
- Build a reproducible data pipeline
- Perform exploratory data analysis (EDA)
- Test hypotheses (e.g., sleep → focus)
- Develop simple predictive models
- Practice statistical reasoning on real-world personal data

## Status
Project initialised. Data schema design in progress.
